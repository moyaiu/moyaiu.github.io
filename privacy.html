usage of data

The bot may use stored data, as defined below, for different features including but not limited to: Leveling, Social Credit and Server Settings. No usage of data outside of the aformentioned cases will happen and the data is not shared with anyone.

Stored Info

The bot stores the following information:

  • guildID containing the server ID this ID will be stored once the bot joined, or after the level/social credit system is enabled.
  • userID containing the userID this ID will be stored after a message was sent with the level/socialcredit system enabled.

    • Other non-critical information may be stored, like which settings are enabled for a guild

    Temporarely stored Information

    The bot may Temporarily get some information when using a commands(like a users current presence). After the execution of a command finished, this data will be dropped.

    Removal of Data

    Removal of the data can be requested through the support server, or a dm to one of the staff members (DaSussy#0362,Shid#5259). For servers, only an admin can request a removal.
